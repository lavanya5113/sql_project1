## Project Overview
This project performs an end-to-end data analysis of Tamil songs from Spotify to understand how musical features such as danceability, energy, duration, and release year influence song popularity.

## Tools & Technologies
- Python (Pandas, Matplotlib)
- SQL (MySQL)
- Jupyter Notebook
- GitHub

- ## Dataset
- Spotify song dataset filtered to include only Tamil songs
- Data cleaned by removing duplicates and handling missing values
- Song duration converted from milliseconds to minutes

## Analysis Performed
- Popularity trends across release years
- Danceability vs Popularity
- Energy vs Popularity
- Song duration impact on popularity
- Consistency-based Top 10 artist analysis

## Visualizations
- Line charts for popularity trends
- Bar charts comparing musical features
- Scatter plots for feature relationships

## Key Insights
- Songs longer than 3 minutes show slightly higher popularity
- High danceability and medium energy songs perform better
- Popularity depends on multiple features, not a single factor
- Consistent artists perform better than one-hit artists
